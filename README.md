
## Description

[Nest](https://github.com/nestjs/nest) Library API. Please visit '/doc' to see swagger documentation.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Swagger

Visit `/docs` to see swagger generated routes from app.
